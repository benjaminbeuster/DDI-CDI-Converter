# Setup for PythonAnywhere
1. create virtual environment in console
mkvirtualenv myvirtualenv --python=/usr/bin/python3.9

2. open virtual environment
workon myvirtualenv

3. install packages
pip install -r requirements.txt

4. change working directory and path of virtual environment in web-tab:
/home/BBeuster/
/home/BBEUSTER/.virtualenvs/myvirtualenv

5. Define WSGI configuration file:
/var/www/bbeuster_pythonanywhere_com_wsgi.py
